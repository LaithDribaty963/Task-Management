# توثيق تقدم مشروع إدارة المهام والمشاريع
امة المه
---

## خصائص المشروع

- **اسم المشروع:** نظام إدارة المهام والمشاريع
- **وصف مختصر:** تطبيق ويب متكامل لإدارة وتنظيم المهام والمشاريع مع دعم فرق العمل، إشعارات، وتكامل مع التقويمات.
- **الهدف:** تسهيل متابعة المهام والمشاريع وتحسين الإنتاجية.
- **الفئة المستهدفة:** فرق العمل، الشركات الصغيرة والمتوسطة، الأفراد.

---

## الأدوات والتقنيات المستخدمة

- **اللغات:** TypeScript, JavaScript
- **الإطار الرئيسي:** Next.js
- **مكتبات الواجهة:** React
- **تنسيق وتصميم:** CSS Modules
- **تنظيم الكود:** Modular Structure (مجلدات منفصلة لكل موديول)
- **أدوات الجودة:** ESLint
- **إدارة الحزم:** npm
- **النشر:** Vercel
- **إدارة الإصدارات:** Git
- **أدوات إضافية مقترحة:** NextAuth (للمصادقة)، Jest (للاختبارات)
